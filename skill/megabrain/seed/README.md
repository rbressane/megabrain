# MegaBrain

This private repository is your local-first Markdown brain. Connected agents synchronize it through Git and store durable knowledge as immutable files under `brain/`.

Normal use happens through conversation. To inspect the brain locally, ask a connected agent to open MegaBrain or run:

```bash
python3 skill/megabrain/scripts/megabrain.py browse
```

The repository intentionally begins with zero personal memories. Never store passwords, tokens, private keys, recovery codes, cookies, or raw connection strings.
