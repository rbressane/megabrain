# MegaBrain

This private repository is your local-first canonical Markdown brain. Connected agents synchronize it through Git and store approved durable memories, resources, import coverage, non-sensitive archive objects, policies, and provenance as immutable files under `brain/`.

Normal use happens through conversation. Ask a connected agent to remember, inspect, update, or open MegaBrain. The independently installed MegaBrain runtime performs those operations; this repository contains personal brain data, not executable product code.

The repository intentionally begins with zero personal knowledge. Never store passwords, tokens, private keys, recovery codes, cookies, raw connection strings, or synchronized sensitive bodies/attachments. Private Git is not encryption.
